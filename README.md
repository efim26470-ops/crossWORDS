# Бесконечный кроссворд

Готовый статический проект для GitHub Pages.

## Что внутри

- `index.html` — вся игра и дизайн.
- `manifest.webmanifest` — PWA-настройки.
- `sw.js` — кэширование для работы как мини-приложение.
- `icons/` — иконки для iPhone, Android, браузера и GitHub Pages.

## Функции

- Темы кроссворда: смешанный, профессии, животные, научный, автоспорт, IT, финансы, география, еда.
- Сложности: новичок, нормальный, сложный, эксперт.
- Визуальные темы: классика, неон, океан, лес, лаборатория, автоспорт, закат.
- Светлый/тёмный режим.
- Проверка ответов, подсказки, раскрытие ответов, таймер, очки, уровни.
- Адаптация под iPhone и мобильные экраны.

## Как выгрузить на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи содержимое этой папки в корень репозитория.
3. Открой Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: `main`, folder: `/root`.
6. Сохрани настройки.

После публикации сайт можно открыть с iPhone через Safari и добавить на экран Домой.
